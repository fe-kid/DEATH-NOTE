### 📓 기획 의도

키라가 되어 흉악한 범죄자들을 응징하세요!<br/>
다른 키라들과 순위를 두고 겨루거나 게시판에서 정의에 대해 토론을 벌일 수도 있습니다.<br/>
비뚤어진 정의를 바로잡기 위한 일이라는 것을 기억하세요. 당신이 정의입니다.

### ⚙️ 사용 기술

- 개발 언어 : TypeScript
- 프론트엔드 라이브러리 : React.js
- 상태 관리 : Redux
- 스타일링 : Styled Components
- 백엔드 : Firebase (Auth + Firestore)
